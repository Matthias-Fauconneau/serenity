Enumerator:

Aligns tightly packed widgets.

Center 

Aligns tightly packed widgets.

Even 

Aligns tightly packed widgets.

Spread 

shares space evenly

Share 

allocates minimum sizes and spreads any extra space between widgets

AlignLeft 

allocates minimum sizes and shares any extra space

AlignRight 

For side axis, sets all widgets side size to maximum needed and aligns left/top.

AlignCenter 

For side axis, sets all widgets side size to maximum needed and aligns right/bottom.

Expand 

For side axis, sets all widgets side size to largest hint (or total available if any widget is expanding) and centers.

For side axis, sets all widgets side size to layout available side size
